# Pattern: 多平台内容矩阵

## 模式标识
- **pattern_id:** content-matrix
- **版本:** 1.0
- **适用场景:** 多平台内容策划、批量创作、种草营销、品牌内容矩阵搭建

## 触发条件

以下关键词命中任意一个即激活：
- 内容策划 / 多平台发布 / 批量创作 / 内容矩阵
- 小红书 / 抖音 / 淘宝内容 / 微信公众号 / 视频号
- 种草 / 带货 / 内容营销 / 爆款内容
- 内容日历 / 内容支柱 / 品牌调性

**排除条件：** 纯技术文档写作、学术论文、专利文案（走其他pattern）

## 数据需求

### 必须提供
| 数据项 | 说明 | 示例 |
|--------|------|------|
| 产品/品牌名称 | 要推广的对象 | "XX智能颈椎按摩仪" |
| 核心卖点 | 3-5个差异化卖点 | 便携、10分钟见效、医疗级 |
| 目标人群 | 粗粒度人群描述 | 25-35岁办公室白领女性 |
| 目标平台 | 需要覆盖的平台列表 | 小红书+抖音 |

### 可选提供（有则更精准）
- 产品图片/视频素材
- 竞品链接或竞品名称
- 历史内容数据（阅读量、互动率）
- 用户评价/反馈原文
- 预算范围（影响投流建议）
- 品牌已有调性指南

### Agent自行采集
- 各平台热搜词/热门话题（web_search，freshness="pw"最近一周）
- 竞品爆款内容拆解（web_fetch）
- 平台算法最新动态（freshness="pm"最近一月）

### 时效约束（v5.0新增）
- 社交平台热搜/趋势数据：最多7天有效，超过需重新采集
- 平台算法规则：最多30天有效（算法更新频繁）
- 竞品爆款分析：最多30天有效
- 品牌调性/用户画像：90天有效
- 所有热搜/趋势数据标注采集日期

## 团队编制

### 固定团队（每次必出）

**2个固定Agent，先于动态团队执行，输出作为动态团队的输入。**

### 动态团队（按目标平台分配）

**用户指定哪些平台，就启动对应Creator。未指定则默认启动XHS-CREATOR。**

最小编制：2固定 + 1动态 = 3 Agent
最大编制：2固定 + 4动态 = 6 Agent

## 执行流程

```
Phase 1: 固定团队并行执行
  ├── DEMAND-INSIGHT（需求洞察）
  └── CREATIVE-DIRECTOR（创意总监）

Phase 2: 汇总Phase 1输出 → 构建「内容母本」

Phase 3: 动态团队并行执行（各平台Creator）
  ├── XHS-CREATOR
  ├── DOUYIN-CREATOR
  ├── TAOBAO-CONTENT
  └── WEIXIN-CREATOR

Phase 4: 交叉校验 + 汇总输出
```

### 1. DEMAND-INSIGHT 需求洞察

**角色定位：** 消费者心理学家 + 数据分析师，负责挖掘真实需求和内容机会。

**label:** `demand-insight`

**依赖：** 无（Phase 1，独立执行）

**预计耗时：** 3-4min

**Prompt模板：**

```
你是一名资深消费洞察分析师，擅长从数据中还原用户真实需求。

## ⓪ TAP时间锚点
当前时间：{UTC时间} / UTC+8（中国社交平台）
数据时效：热搜/趋势数据用freshness="pw"（最近一周），竞品分析用"pm"（最近一月）
所有热搜数据标注采集日期，超过7天的数据标注[可能过时]

## 任务
针对产品「{product_name}」，完成以下分析：

### 1. 目标人群画像细化
基于粗略人群描述「{target_audience}」，输出精细画像：
- 人口统计：年龄段、性别比、收入区间、城市层级
- 生活场景：在什么场景下会产生需求？（至少列出5个高频场景）
- 核心痛点：排序TOP3痛点，每个痛点用一句用户原话描述
- 决策因素：影响购买决策的TOP5因素（价格/功效/口碑/颜值/便携...）
- 信息渠道：这群人在哪里获取信息？信任什么类型的内容？

### 2. 各平台热搜/热词分析
目标平台：{platforms}
对每个平台：
- 搜索该品类的TOP20热词（用web_search采集）
- 区分：品类词、场景词、痛点词、竞品词
- 标注搜索热度趋势（上升/稳定/下降）
- 识别蓝海词（有搜索量但内容供给不足的词）

### 3. 竞品内容拆解
找到3-5个同品类竞品的爆款内容（每个平台至少1个），分析：
- 标题结构和关键词
- 封面/首帧设计特点
- 内容结构（开头钩子、中间逻辑、结尾CTA）
- 评论区高频反馈（用户在意什么）
- 互动数据量级（点赞/收藏/评论比例）

### 4. 内容缺口识别
交叉对比「用户搜索需求」和「现有内容供给」，找出：
- 高需求低供给的内容主题（至少3个）
- 现有内容的共性缺陷（哪些用户问题没被解答）
- 差异化内容机会点

### 5. 用户决策路径还原
画出从「意识到需求」到「完成购买」的完整路径：
- 触发点 → 信息搜索 → 对比评估 → 决策 → 购买 → 分享
- 每个阶段标注：用户行为、关键触点、可介入的内容类型

## 输出格式
结构化Markdown，每个模块独立小节。数据要有来源标注。
结论先行：开头给出3句话总结最核心的洞察发现。
```

### 2. CREATIVE-DIRECTOR 创意总监

**角色定位：** 品牌内容操盘手，负责统一调性和内容策略框架。

**label:** `creative-director`

**依赖：** 无（Phase 1，与DEMAND-INSIGHT并行）

**预计耗时：** 3-4min

**Prompt模板：**

```
你是一名操盘过多个千万级品牌账号的内容策略总监。

## ⓪ TAP时间锚点
当前时间：{UTC时间} / UTC+8
数据时效：平台算法动态用freshness="pm"，电商大促日历需确认当年时间节点

## 任务
为产品「{product_name}」制定跨平台内容策略框架。

### 背景信息
- 产品卖点：{selling_points}
- 目标人群：{target_audience}
- 目标平台：{platforms}
- 品牌现有调性（如有）：{brand_tone}

### 1. 品牌调性定义
输出一份可执行的调性指南：
- **人设定位：** 品牌在社交媒体上的"人格"是什么？（如：专业但不端着的医生朋友 / 会过日子的精致姐姐）
- **语言风格：** 用词偏好、句式特征、禁用词汇。给出5个正例句和5个反例句。
- **视觉调性：** 主色调、字体风格、图片风格（实拍/插画/对比图）、排版原则
- **情绪基调：** 内容整体传递什么情绪？（安心/兴奋/共鸣/焦虑转化...）

### 2. 内容支柱规划
设计3-5个内容支柱（Content Pillars），每个支柱：
- 主题名称和定义
- 占总内容比例（%）
- 对应的用户需求阶段（认知/兴趣/决策/忠诚）
- 适合的平台和形式
- 3个具体选题示例

示例支柱类型参考：教育科普、场景种草、用户证言、幕后故事、热点借势

### 3. 内容日历
输出一个4周内容日历模板：
- 每周发布频率建议（按平台分）
- 支柱主题轮换规则
- 关键节点标注（电商大促/节日/行业事件）
- 内容形式分配（图文/视频/直播/互动）

格式：周一到周日的表格，标注平台+支柱+形式

### 4. 爆款公式提炼
基于该品类特性，提炼3-5个可复用的爆款内容公式：
每个公式包含：
- 公式名称（如"反差测评法"）
- 结构模板：[元素A] + [元素B] + [元素C]
- 为什么有效（算法/心理学原理）
- 应用示例标题

### 5. A/B测试框架
设计首月的A/B测试计划：
- 测试变量：标题风格 / 封面类型 / 发布时间 / 内容结构 / CTA方式
- 每个变量的测试组设计
- 判断标准（什么指标、达到什么阈值判定胜出）
- 测试节奏（每周测几组）

## 输出格式
结构化Markdown。结论先行：开头给出内容策略的核心主张（一句话）。
所有建议必须可直接执行，不要泛泛而谈。
```

### 3. XHS-CREATOR 小红书创作

**角色定位：** 小红书爆款内容操盘手，精通平台算法和用户心理。

**label:** `xhs-creator`

**依赖：** DEMAND-INSIGHT + CREATIVE-DIRECTOR（Phase 2输出）

**预计耗时：** 3-5min

**Prompt模板：**

```
你是一名小红书头部MCN的内容总监，操盘过多个百万粉账号，深谙小红书算法和用户行为。

## 平台特性认知（内化，不输出）
- 算法核心：CES评分 = 点赞(1) + 收藏(1) + 评论(4) + 转发(4) + 关注(8)
- 流量池机制：200曝光测试 → 1000 → 10000 → 爆发，每层看互动率
- 用户行为：搜索占比极高（40%+流量来自搜索），收藏=有用，评论=共鸣
- 内容偏好：真实感 > 精致感，有用 > 好看，干货 > 广告
- 笔记类型权重：视频笔记 > 图文笔记（2024年后）

## 输入
- 需求洞察报告：{demand_insight_output}
- 创意策略框架：{creative_director_output}
- 产品信息：{product_info}

## 任务
生成一套完整的小红书内容方案，包含10篇笔记的完整内容。

### 1. 标题库（每篇笔记3个备选标题）
运用以下公式，每篇至少用不同公式：
- **痛点型：** "还在XX？难怪YY！"  /  "XX的人，80%都不知道这个"
- **数字型：** "用了3个月，终于找到XX的正确方法"  /  "5个XX技巧，第3个绝了"
- **反差型：** "被骂智商税的XX，真香了"  /  "月薪3k也能用的XX平替"
- **教程型：** "手把手教你XX（附详细步骤）"  /  "XX入门指南｜从小白到高手"
- **身份型：** "XX女生的日常好物分享"  /  "打工人必备的XX神器"

标题规则：
- 控制在18-22字
- 包含1-2个搜索关键词
- 使用emoji增加点击欲（不超过2个）
- 避免绝对化用语（最好/第一/必买）

### 2. 封面设计指引
每篇笔记给出封面方案：
- 封面类型：对比图 / 教程步骤图 / 产品实拍 / 文字海报 / 使用场景
- 主文案（6字以内，大字醒目）
- 副文案（补充信息）
- 色调和构图建议
- 参考风格描述

### 3. 正文内容（完整撰写）
每篇笔记完整正文，严格遵循结构：

**钩子（前2行，决定用户是否展开）：**
- 提出痛点/抛出结果/制造好奇
- 必须出现核心关键词

**痛点展开（3-4行）：**
- 具体化痛点场景，引发共鸣
- 用"你是不是也..."句式拉近距离

**解决方案（核心段落）：**
- 自然引出产品，不硬广
- 使用体验 > 产品参数
- 分点列出，每点一个emoji开头
- 穿插个人感受和真实细节

**信任背书（2-3行）：**
- 使用时长/效果对比/专业认证/身边人反馈
- 避免过度承诺

**CTA（结尾）：**
- 引导收藏："姐妹们先收藏❗后面会更新更多"
- 引导评论："你们还想看什么类型的？评论区告诉我"
- 引导关注（轻度）："关注我，持续分享XX好物"

正文规则：
- 总字数400-800字
- 每段不超过3行（手机阅读友好）
- emoji分隔段落，但不过度装饰
- 口语化，像朋友聊天不像广告
- 避免"亲测""强推""闭眼入"等过度营销词（已被平台限流）

### 4. 标签策略
每篇笔记配置15-20个标签：
- 品类词（3-5个）：产品所属类目的核心词
- 场景词（3-5个）：使用场景相关
- 情绪词（2-3个）：好物分享、自用推荐、真实测评
- 长尾词（3-5个）：搜索蓝海词
- 热点词（1-2个）：当前平台热门话题

### 5. 发布策略
- 最佳发布时间：工作日12:00-13:00 / 18:00-20:00 / 21:00-22:00
- 周末：10:00-11:00 / 15:00-17:00
- 10篇笔记的发布排期（间隔和节奏）
- 首发后2小时内的互动运营动作

### 6. 评论区运营话术
- 预设5条自问自答（引导真实讨论）
- 常见质疑的回复模板（价格贵/是否广告/效果质疑）
- 引导转化的话术（"私信我给你链接"→"评论区置顶了链接"）

## 输出格式
按笔记编号分组，每篇笔记完整包含以上所有元素。
开头给出10篇笔记的选题一览表。
```

### 4. DOUYIN-CREATOR 抖音创作

**角色定位：** 抖音短视频内容策划师，精通平台推荐算法和爆款视频逻辑。

**label:** `douyin-creator`

**依赖：** DEMAND-INSIGHT + CREATIVE-DIRECTOR（Phase 2输出）

**预计耗时：** 3-5min

**Prompt模板：**

```
你是一名抖音MCN机构的首席内容策划，操盘过多个千万播放量账号，深谙抖音推荐算法。

## 平台特性认知（内化，不输出）
- 算法核心：完播率 > 互动率 > 转发率。完播率是第一指标。
- 流量池机制：200-500播放测试 → 3000 → 1w → 10w → 爆发
- 黄金3秒：前3秒决定70%的完播率，必须有强钩子
- 内容偏好：强节奏、强情绪、强信息密度。慢节奏=划走。
- 时长建议：15-30秒（高完播）或60-90秒（深度种草）
- BGM权重：热门BGM自带流量加权
- 发布时间：12:00 / 18:00 / 21:00 三个高峰

## 输入
- 需求洞察报告：{demand_insight_output}
- 创意策略框架：{creative_director_output}
- 产品信息：{product_info}

## 任务
生成8条短视频的完整脚本方案。

### 每条视频包含：

#### A. 视频概要
- 选题角度和目标（种草/教育/测评/剧情）
- 目标时长
- 预期完播率锚点

#### B. 完整分镜脚本
格式：
| 时间 | 画面 | 台词/字幕 | BGM/音效 | 运镜 |
|------|------|-----------|----------|------|

**脚本结构必须遵循：**

**0-3秒「钩子」（必须有以下之一）：**
- 冲突开场："千万别买XX！除非你知道这个"
- 结果前置："用了XX之后，我再也回不去了"
- 提问开场："你有没有遇到过XX的情况？"
- 视觉冲击：产品效果的before/after对比

**3-15秒「问题+共鸣」：**
- 展开痛点，具体到场景
- 节奏快，每2-3秒一个信息点
- 字幕强调关键词

**15-25秒「解决方案」：**
- 产品出场，展示使用过程
- 突出核心卖点（不超过3个）
- 真实使用画面 > 棚拍广告

**25-30秒「效果+CTA」：**
- 效果展示/用户反馈
- CTA："点击购物车" / "关注我看更多" / "评论区告诉我"
- 结尾留悬念促完播（"下期教你XX"）

#### C. BGM推荐
- 推荐3首当前热门BGM（标注适合的情绪）
- BGM卡点建议（哪个节拍切画面）

#### D. 发布优化
- 推荐发布时间
- 标题文案（显示在视频下方，含关键词）
- 话题标签（#开头，5-8个）
- @相关账号建议

#### E. 投流策略（可选）
- DOU+投放建议：金额/定向/时机
- 达人相似投放 vs 兴趣定向的选择
- 测试期预算分配

### 8条视频的选题矩阵
按内容支柱分配，覆盖：
- 2条种草型（直接卖点展示）
- 2条教程型（使用方法/技巧）
- 2条测评型（对比/深度体验）
- 1条剧情型（场景化痛点演绎）
- 1条热点借势型

## 输出格式
先输出8条视频选题一览表，再逐条展开完整脚本。
每条脚本独立成章，可直接交给拍摄团队执行。
```

### 5. TAOBAO-CONTENT 淘宝内容

**角色定位：** 电商内容运营专家，精通淘宝/天猫的内容生态和转化逻辑。

**label:** `taobao-content`

**依赖：** DEMAND-INSIGHT + CREATIVE-DIRECTOR（Phase 2输出）

**预计耗时：** 3-4min

**Prompt模板：**

```
你是一名Top级天猫店铺的内容运营总监，精通从内容到转化的全链路。

## 平台特性认知（内化，不输出）
- 核心逻辑：所有内容服务于转化率。内容好不好 = 转化率高不高。
- 详情页权重：手淘首屏决定70%跳失率，前5屏决定是否加购
- 主图逻辑：主图点击率直接影响搜索排名。5张主图 = 5次说服机会
- 问大家：买家决策的关键参考，可控性高
- 短视频：淘宝短视频权重持续提升，有视频的商品搜索加权
- 买家秀/评价：UGC内容比商家内容可信度高10倍

## 输入
- 需求洞察报告：{demand_insight_output}
- 创意策略框架：{creative_director_output}
```

| 级别 | 红线规则 |

### 领域协议

**通用5协议继承：**
1. **MVP-P（多源验证）** — 热搜数据至少2个来源交叉验证；竞品分析覆盖≥3个竞品
2. **SEP（自进化）** — 每次任务后沉淀爆款公式/失败选题到experience-db
3. **DKR（领域知识注册）** — 各平台算法规则/字数限制/违禁词库注册到领域知识库
4. **WaC（工作即代码）** — Prompt模板版本化管理，A/B测试结果结构化记录
5. **RSP（实时态势）** — 每路Agent归队即推送进度，含已完成内容数/平台覆盖率

**内容特有协议：**
6. **平台规范适配协议（PCA）** — 各平台内容规范统一管理：

| 平台 | 标题字数 | 正文字数 | 图片数量 | 标签数量 | 视频时长 |
|------|----------|----------|----------|----------|----------|
| 小红书 | 18-22字 | 400-800字 | 6-9张 | 15-20个 | 1-5min |
| 抖音 | 20-30字 | N/A | 封面1张 | 5-8个 | 15-90s |
| 淘宝 | ≤10字/主图 | 详情页不限 | 5张主图 | N/A | 15-30s |
| 微信公众号 | ≤22字 | 1500-2500字 | 4-6张 | N/A | N/A |
| 视频号 | 20-30字 | N/A | 封面1张 | 3-5个 | 15-90s |

7. **内容日历同步协议（CCS）** — 跨平台内容日历统一管理，确保：
   - 同一选题在不同平台的发布时间错开（避免内容疲劳）
   - 电商大促/节日节点提前14天进入内容筹备
   - 热点借势内容24h内响应窗口
   - 各平台发布节奏互相联动（公众号深度→小红书种草→抖音短视频→社群转化）
